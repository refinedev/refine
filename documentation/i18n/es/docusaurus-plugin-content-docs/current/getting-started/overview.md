## ¿Qué es Refine?

**Refine** es un meta-framework de React para aplicaciones web con muchas operaciones CRUD. Está pensado para casos empresariales como herramientas internas, paneles de administración, dashboards y aplicaciones B2B.

Sus hooks y componentes principales reducen trabajo repetitivo al ofrecer soluciones estándar para **autenticación**, **control de acceso**, **routing**, **networking**, **gestión de estado** e **i18n**.

La arquitectura headless de Refine separa la lógica de negocio de la interfaz y del enrutamiento. Esto permite integrarlo con diseños propios, frameworks de UI como Tailwind CSS, Ant Design, Material UI, Mantine y Chakra UI, y plataformas como Next.js, Remix, React Native o Electron.

## ¿Por qué usar Refine?

Refine se sitúa entre crear todo desde cero y adoptar una solución low-code/no-code. Mantiene flexibilidad y tecnologías abiertas, pero aporta convenciones para evitar retrasos, deuda técnica, problemas de mantenimiento y código difícil de estandarizar.

Las soluciones low-code reducen parte de ese trabajo, pero pueden traer bloqueo con proveedores, pocas opciones de personalización y una peor experiencia para desarrolladores. Refine ofrece una base productiva sin cerrar el proyecto a una arquitectura rígida.

## Casos de uso

**Refine** destaca en aplicaciones intensivas en datos, por ejemplo:

- Paneles de administración.
- Dashboards operativos.
- Herramientas internas.
- Aplicaciones B2B con flujos CRUD.

## Funciones clave

- Devtools para inspeccionar la aplicación y obtener información útil.
- Conectores para REST API, GraphQL, NestJS CRUD, Airtable, Strapi, Supabase, Hasura, Appwrite, Firebase, Nestjs-Query y Directus.
- Soporte SSR con Next.js y Remix.
- Generación de interfaces CRUD a partir de la estructura de la API.
- Gestión de estado y mutaciones con TanStack Query.
- Providers para autenticación, autorización, notificaciones, tiempo real y auditoría.
