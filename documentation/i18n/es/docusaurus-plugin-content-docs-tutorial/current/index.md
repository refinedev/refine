El tutorial de Refine guía la construcción de una aplicación CRUD completa. A lo largo del recorrido se configuran resources, providers, routing, formularios, tablas y flujos de autenticación.

## Qué aprenderás

- Crear una aplicación Refine desde una base limpia.
- Conectar un data provider a una API.
- Definir resources y rutas.
- Construir páginas de listado, creación, edición y detalle.
- Añadir autenticación, autorización e i18n cuando el proyecto lo necesite.

## Cómo usar esta traducción

Los nombres de paquetes, imports, comandos y fragmentos de código se mantienen en inglés para coincidir con la API real. Las explicaciones y notas de apoyo se localizan al español para facilitar el aprendizaje.
