**Refine** работает в любой среде, где можно запускать **React**, включая Vite, Next.js, Remix и CRA.

Пакеты можно установить вручную, но рекомендуемый путь — browser scaffolder или CLI. Оба варианта позволяют выбрать framework, UI, data provider, authentication и i18n.

## Использование CLI

```sh
npm create refine-app@latest
```

Ответьте на вопросы, перейдите в созданную папку, установите зависимости при необходимости и запустите dev server командой, которую покажет CLI.

## Использование браузера

Browser scaffolder предлагает те же ключевые настройки, что и CLI, и позволяет preview результата перед скачиванием.
