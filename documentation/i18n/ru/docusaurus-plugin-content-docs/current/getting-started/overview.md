## Что такое Refine?

**Refine** — это React meta-framework для веб-приложений с большим количеством CRUD-операций. Он подходит для внутренних инструментов, admin panels, dashboards и B2B-приложений.

Основные hooks и components Refine ускоряют разработку, предлагая стандартные решения для **authentication**, **access control**, **routing**, **networking**, **state management** и **i18n**.

Headless-архитектура отделяет бизнес-логику от UI и маршрутизации. Поэтому Refine можно использовать с собственным дизайном, Tailwind CSS, Ant Design, Material UI, Mantine, Chakra UI, Next.js, Remix, React Native или Electron.

## Почему Refine?

Refine находится между разработкой полностью from scratch и low-code/no-code. Он сохраняет гибкость открытых технологий, но добавляет соглашения, которые снижают задержки, технический долг и проблемы сопровождения.

## Сценарии использования

Refine особенно полезен для data-intensive приложений: admin panels, dashboards, internal tools и B2B CRUD flows.

## Ключевые возможности

- Devtools для анализа приложения.
- Connectors для REST API, GraphQL, NestJS CRUD, Airtable, Strapi, Supabase, Hasura, Appwrite, Firebase, Nestjs-Query и Directus.
- SSR с Next.js и Remix.
- Генерация CRUD UI из структуры API.
- State management и mutations через TanStack Query.
- Providers для auth, permissions, notifications, realtime и audit logs.
