Tutorial Refine ведет через создание полноценного CRUD application. В процессе настраиваются resources, providers, routing, forms, tables и authentication flows.

## Что вы изучите

- Создание Refine app с чистой базы.
- Подключение data provider к API.
- Определение resources и routes.
- Создание страниц list, create, edit и show.
- Добавление authentication, authorization и i18n по необходимости.

Package names, imports, commands и code examples остаются на английском, чтобы совпадать с реальной API; пояснения локализованы на русский.
