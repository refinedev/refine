## Exemple i18n avec Next.js

Cet exemple montre comment combiner **Refine**, Next.js et l'internationalisation pour adapter routes, rendu et contenu à la langue de l'utilisateur.

## Tester localement

```bash
npm create refine-app@latest -- --example i18n-nextjs
```

## Points clés

- Configuration des locales dans Next.js.
- Intégration de l'i18n provider avec `<Refine />`.
- Traduction de la navigation et des actions CRUD.
- Compatibilité avec le routing et le rendering de Next.js.
