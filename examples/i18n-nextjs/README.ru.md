## Пример i18n с Next.js

Этот пример показывает, как объединить **Refine**, Next.js и internationalization, чтобы адаптировать routes, rendering и content к языку пользователя.

## Запуск локально

```bash
npm create refine-app@latest -- --example i18n-nextjs
```

## Ключевые моменты

- Настройка locales в Next.js.
- Интеграция i18n provider с `<Refine />`.
- Перевод navigation и CRUD actions.
- Совместимость с routing и rendering Next.js.

[Открыть пример i18n-nextjs](https://codesandbox.io/embed/github/refinedev/refine/tree/main/examples/i18n-nextjs?view=preview&theme=dark&codemirror=1)
