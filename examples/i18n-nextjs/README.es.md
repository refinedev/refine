## Ejemplo i18n con Next.js

Este ejemplo enseña cómo combinar **Refine**, Next.js e internacionalización. Es útil cuando necesitas rutas, renderizado y contenido adaptados al idioma del usuario.

## Probar localmente

```bash
npm create refine-app@latest -- --example i18n-nextjs
```

## Puntos importantes

- Configuración de locales en Next.js.
- Integración del provider de i18n con `<Refine />`.
- Traducción de elementos de navegación y acciones CRUD.
- Compatibilidad con rutas y rendering de Next.js.
