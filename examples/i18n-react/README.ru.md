## Пример i18n с React

Этот пример показывает, как использовать **Refine** с internationalization в React application. CRUD logic остается в Refine, а translations подключаются через подходящий provider.

## Запуск локально

```bash
npm create refine-app@latest -- --example i18n-react
```

## Что проверить

- Конфигурацию i18n provider.
- Смену языка в интерфейсе.
- Перевод menus, actions и visible text.
- Сохранение resources, routes и APIs в code.
