## Exemple i18n avec React

Cet exemple montre comment utiliser **Refine** avec l'internationalisation dans une application React. La logique CRUD reste gérée par Refine et les traductions sont fournies par un provider adapté.

## Tester localement

```bash
npm create refine-app@latest -- --example i18n-react
```

## À vérifier

- Configuration de l'i18n provider.
- Changement de langue depuis l'interface.
- Traduction des menus, actions et textes visibles.
- Conservation des resources, routes et APIs dans le code.
