## Ejemplo i18n con React

Este ejemplo muestra cómo usar **Refine** con internacionalización en una aplicación React. Mantiene la lógica CRUD de Refine y conecta las traducciones mediante un provider compatible con la aplicación.

## Probar localmente

```bash
npm create refine-app@latest -- --example i18n-react
```

## Qué revisar

- Configuración del provider de i18n.
- Cambio de idioma desde la interfaz.
- Traducción de menús, acciones y textos visibles.
- Conservación de nombres de resources, rutas y APIs en el código.
