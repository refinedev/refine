## Template Material UI

Ce template initialise une application **Refine** avec Material UI. C'est une base pratique pour panels d'administration, dashboards et outils internes qui ont besoin de composants accessibles et personnalisables.

## Tester localement

```bash
npm create refine-app@latest -- --example template-material-ui
```

## Inclus

- Intégration Refine avec Material UI.
- Structure prête pour resources et pages CRUD.
- Point de départ pour auth, permissions, notifications et i18n.

Après création, configurez votre data provider et définissez les resources principaux de l'application.
