## Template Material UI

Этот template создает приложение **Refine** с Material UI. Он подходит как основа для admin panels, dashboards и internal tools, которым нужны accessible и customizable components.

## Запуск локально

```bash
npm create refine-app@latest -- --example template-material-ui
```

## Что включено

- Интеграция Refine с Material UI.
- Структура для resources и CRUD pages.
- База для auth, permissions, notifications и i18n.

После создания настройте data provider и определите основные resources приложения.
