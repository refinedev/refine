## Plantilla Material UI

Esta plantilla inicia una aplicación **Refine** con Material UI. Es una buena base para paneles administrativos, dashboards y herramientas internas que necesitan componentes accesibles y personalizables.

## Probar localmente

```bash
npm create refine-app@latest -- --example template-material-ui
```

## Qué incluye

- Integración de Refine con Material UI.
- Estructura preparada para resources y páginas CRUD.
- Punto de partida para añadir autenticación, permisos, notificaciones e i18n.

## Siguiente paso

Después de crear la plantilla, configura tu data provider y define los resources principales de la aplicación.
