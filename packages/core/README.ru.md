# Refine CORE

**Refine CORE** — React meta-framework для web applications с большим количеством CRUD. Он служит основой для internal tools, admin panels, dashboards и B2B applications.

Refine CORE использует headless architecture: data, routing, auth, permissions, notifications, state и i18n отделены от визуального слоя.

## Создание проекта

```sh
npm create refine-app@latest my-refine-app
```

## Почему это удобно

- Совместимо с любым design или UI framework.
- Hooks и providers для типичных enterprise patterns.
- Работает с REST, GraphQL и множеством backend services.
- Убирает повторяющуюся работу без навязывания fixed UI.
