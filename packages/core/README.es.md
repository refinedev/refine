# Refine CORE

**Refine CORE** es un meta-framework de React para aplicaciones web con muchas operaciones CRUD. Sirve como base para herramientas internas, paneles administrativos, dashboards y aplicaciones B2B.

Refine CORE ofrece una arquitectura headless: la lógica de datos, routing, autenticación, autorización, notificaciones, estado e i18n se mantiene separada de la capa visual.

## Crear un proyecto

```sh
npm create refine-app@latest my-refine-app
```

## Por qué usarlo

- Permite usar cualquier diseño o framework de UI.
- Incluye hooks y providers para patrones empresariales comunes.
- Funciona con REST, GraphQL y múltiples servicios backend.
- Reduce trabajo repetitivo sin encerrar la aplicación en una UI fija.
